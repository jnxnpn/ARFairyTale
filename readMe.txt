浸入式AR童话亲子互动平台
			By 潘进新，李元博，房逸铭，刘泽怡

 大亮点

· 拥有scrapy爬虫支持的海量童话数据库

· 利用自然语言处理tf-idf算法进行故事关键词提取，更好的贴近客户需求；利用机器学习推荐算法，将最好的故事推荐给孩子

· Unity3D构建童话世界，让孩子不仅仅可以听故事，更可以“玩”故事；孩子可以改变故事发展走向，并且通过身临其境的体验，了解到童话故事背后的人生哲理

· 微信小程序构建温馨家长端，让家长自定义故事，并可以将好的故事在专属社区中进行分享

· 拥有超强的反馈学习机制，我们会把孩子在游戏过程中的各项指标传入数据库进行存档，通过对孩子的体验进行分析筛选故事，帮助孩子培养正确的人生观，价值观；家长的反馈也将最为重要指标被录入数据库，以供长期项目优化升级使用。
